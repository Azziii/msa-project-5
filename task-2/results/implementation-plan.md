# План реализации и конфигурации решения

## Шаг 1. Разработка сервиса генерации прайс-листов

Создать отдельный контейнеризированный сервис.

Технологии:

* Java Spring Boot или Python
* PostgreSQL Driver
* CSV/XLS Export Library

---

## Шаг 2. Реализация SQL-запросов

Подготовить запрос объединения данных:

* products
* categories
* clients
* client_prices

Результат должен содержать:

* идентификатор товара
* наименование
* категорию
* цену клиента
* дополнительные атрибуты товара

---

## Шаг 3. Формирование файлов

Для каждого клиента:

1. Выполнить выборку данных.
2. Сгенерировать CSV.
3. При необходимости сформировать XLS.
4. Сохранить файл во временный каталог.

---

## Шаг 4. Контейнеризация

Подготовить Dockerfile.

Собрать образ:

```bash
docker build -t price-exporter .
```

Разместить образ в Container Registry.

---

## Шаг 5. Настройка Kubernetes CronJob

Создать CronJob:

```cron
0 6 * * *
```

Настроить:

* restartPolicy: Never
* CPU limits
* Memory limits
* ServiceAccount

---

## Шаг 6. Подключение объектного хранилища

Использовать:

* Amazon S3
  или
* MinIO

После генерации файл загружается в Bucket.

---

## Шаг 7. Мониторинг и логирование

Настроить:

* Prometheus
* Grafana
* ELK

Контролировать:

* успешность выполнения Job
* время выполнения
* количество сформированных файлов
* ошибки выгрузки

---

## Шаг 8. Тестирование

Проверить:

* корректность SQL-запросов
* генерацию CSV
* генерацию XLS
* расписание запуска
* повторный запуск CronJob
* обработку ошибок БД
* обработку ошибок Object Storage

---

## Шаг 9. Ввод в эксплуатацию

Выполнить деплой в production-кластер Kubernetes.

Настроить алертинг на неуспешное выполнение CronJob.
